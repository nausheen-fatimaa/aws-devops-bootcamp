## Lab Performed

### Created Project Directory

```bash
mkdir project
cd project
```

### Created Files

```bash
touch employee.txt
touch salary.txt
touch department.txt
```

### Added Data

```bash
echo "Nausheen" > employee.txt
echo "50000" > salary.txt
echo "DevOps" > department.txt
```

### Copied File

```bash
cp employee.txt employee_backup.txt
```

### Renamed File

```bash
mv salary.txt salary_2026.txt
```

### Created Backup Folder

```bash
mkdir Backup
mv employee_backup.txt Backup/
```
