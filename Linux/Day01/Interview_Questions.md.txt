# Linux Interview Questions

## 1. Difference between cp and mv?

cp copies files while mv moves or renames files.

## 2. Difference between ls and ls -l?

ls lists file names.

ls -l shows detailed information like permissions, owner, size and date.

## 3. What is pwd?

Print Working Directory.

## 4. Difference between Absolute and Relative Path?

Absolute path starts from the root directory (/).

Relative path starts from the current working directory.

## 5. What is mkdir?

Creates a new directory.
