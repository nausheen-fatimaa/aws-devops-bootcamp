# Git Commands Documentation

---

## 1. git init

Purpose:
Initializes a new Git repository.

Syntax

git init

Example

git init

---

## 2. git status

Purpose:
Displays the current state of the repository.

Syntax

git status

Example

git status

---

## 3. git add

Purpose:
Moves files from Working Directory to Staging Area.

Syntax

git add filename

or

git add .

Example

git add .

---

## 4. git commit

Purpose:
Creates a permanent snapshot of staged files.

Syntax

git commit -m "Commit Message"

Example

git commit -m "Initial Commit"

---

## 5. git log

Purpose:
Shows commit history.

Syntax

git log

Example

git log

---

## 6. git log --oneline

Purpose:
Shows short commit history.

Syntax

git log --oneline

---

## 7. git branch

Purpose:
Lists all branches.

Syntax

git branch

---

## 8. git branch feature-login

Purpose:
Creates a new branch.

Syntax

git branch branch_name

Example

git branch feature-login

---

## 9. git checkout

Purpose:
Switches branches.

Syntax

git checkout branch_name

Example

git checkout feature-login

---

## 10. git switch

Purpose:
Alternative command to switch branches.

Syntax

git switch branch_name

---

## 11. git merge

Purpose:
Merges another branch into current branch.

Syntax

git merge branch_name

---

## 12. git restore

Purpose:
Restores modified files.

Syntax

git restore filename

---

## 13. git rm

Purpose:
Removes tracked files.

Syntax

git rm filename

---

## 14. git diff

Purpose:
Displays file changes.

Syntax

git diff

---

## 15. git clone

Purpose:
Downloads an existing repository.

Syntax

git clone repository_url

---

## 16. git push

Purpose:
Uploads commits to GitHub.

Syntax

git push origin main

---

## 17. git pull

Purpose:
Downloads latest changes.

Syntax

git pull origin main

---

## 18. git fetch

Purpose:
Fetches remote changes without merging.

Syntax

git fetch

---

## 19. git remote -v

Purpose:
Displays connected remote repositories.

Syntax

git remote -v

---

## 20. git branch -d

Purpose:
Deletes a branch.

Syntax

git branch -d branch_name

Example

git branch -d feature-login
