# Linux Day 3 Assignment

## Objective

To learn Linux process management, service management, package management, log analysis, and cron jobs.

## Practical Tasks Completed

✅ Monitored running processes using ps, ps -ef, ps aux and top.

✅ Created and managed background processes.

✅ Learned to identify and terminate processes using kill.

✅ Managed Linux services using systemctl.

✅ Installed, updated, and removed software packages using apt.

✅ Explored Linux log files in /var/log.

✅ Viewed authentication and system logs.

✅ Practiced log monitoring using tail and grep.

✅ Learned how to create and manage cron jobs.

## Challenges Faced

* Understanding the difference between ps, ps aux, ps -ef, and top.
* Learning how Linux services are managed using systemctl.
* Understanding how cron jobs execute automatically.

## Learning Outcome

After completing this lab, I can monitor Linux processes, manage services, install software packages, analyze log files, and automate repetitive tasks using cron jobs.
