# Linux Day 05 - Networking Fundamentals & AWS VPC

## Objective

The objective of Day 05 was to understand Linux networking concepts and build a custom AWS networking environment using Amazon VPC. This included learning how devices communicate over a network, understanding IP addressing, DNS, protocols, ports, and deploying an EC2 instance inside a custom VPC.
