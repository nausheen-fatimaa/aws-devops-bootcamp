# AWS Day 01 – Cloud Fundamentals

## Objective

The objective of this lab was to understand the fundamentals of AWS Cloud and become familiar with the AWS Management Console.

---

## Topics Covered

- Introduction to Cloud Computing
- Benefits of Cloud Computing
- Cloud Deployment Models
- Service Models (IaaS, PaaS, SaaS)
- AWS Global Infrastructure
- AWS Regions
- Availability Zones
- Edge Locations
- AWS Free Tier
- AWS Shared Responsibility Model
- AWS Management Console
- Billing Dashboard

---

## Practical Activities Performed

- Logged into AWS Console
- Explored AWS Services
- Switched AWS Regions
- Explored EC2 Dashboard
- Explored IAM Dashboard
- Explored S3 Dashboard
- Explored VPC Dashboard
- Explored Billing Dashboard
- Understood AWS Global Infrastructure

---

## Key Learnings

- AWS provides cloud services on a pay-as-you-go model.
- AWS Regions consist of multiple Availability Zones.
- Edge Locations improve content delivery speed.
- AWS follows the Shared Responsibility Model.
- The Free Tier helps beginners learn AWS without significant costs.

---

## Skills Gained

- AWS Console Navigation
- Understanding AWS Architecture
- Cloud Computing Basics
- AWS Global Infrastructure
- Billing Awareness

---

## Conclusion

AWS Cloud Fundamentals provide the foundation for all AWS services. Understanding cloud concepts and AWS infrastructure is essential before working with compute, networking, storage, and security services.
