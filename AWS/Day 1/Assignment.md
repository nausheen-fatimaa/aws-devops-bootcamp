# AWS Day 01 Assignment

## Q1. What is Cloud Computing?

Cloud computing is the delivery of computing services such as servers, storage, networking, databases, and software over the internet on demand. Users pay only for the resources they consume.

---

## Q2. Why is Cloud Computing better than Traditional Infrastructure?

Cloud computing eliminates the need to purchase and maintain physical servers. It offers scalability, flexibility, high availability, global accessibility, and cost savings through a pay-as-you-go pricing model.

---

## Q3. Explain Public, Private, and Hybrid Cloud.

Public Cloud:
Infrastructure is owned and managed by a cloud provider like AWS.

Private Cloud:
Infrastructure is dedicated to a single organization.

Hybrid Cloud:
A combination of public and private cloud environments working together.

---

## Q4. Differentiate IaaS, PaaS, and SaaS.

IaaS:
Provides virtual infrastructure such as servers, storage, and networking.

Example:
Amazon EC2

PaaS:
Provides a platform to develop and deploy applications.

Example:
AWS Elastic Beanstalk

SaaS:
Provides ready-to-use software over the internet.

Example:
Gmail

---

## Q5. What is an AWS Region?

A Region is a physical geographical location where AWS operates multiple Availability Zones.

---

## Q6. What is an Availability Zone?

An Availability Zone is one or more isolated data centers within an AWS Region.

---

## Q7. What are Edge Locations?

Edge Locations are global locations used by AWS CloudFront to cache content closer to users, reducing latency.

---

## Q8. Explain the Shared Responsibility Model.

AWS secures the cloud infrastructure, while customers are responsible for securing their applications, operating systems, identities, and data.

---

## Q9. Why is AWS Free Tier useful?

It allows beginners to practice AWS services within specified usage limits without significant cost.

---
